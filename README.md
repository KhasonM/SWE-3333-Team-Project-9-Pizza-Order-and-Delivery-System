# MOM & POP’S Pizzaria’s Pizza System

[cite_start]This is the SWE 3313 team project to design a pizza order and delivery system for a new "Mom and Pop" style pizza shop[cite: 197, 198]. [cite_start]The system will manage new customers, take orders, process payments, and contain the restaurant menu[cite: 200].

## Project Structure

* `/assets`: Contains GUI images (icons, menu item photos, etc.)
* `/db`: Contains the Microsoft Access database file (`MomAndPops.accdb`).
* [cite_start]`/deliverables`: Contains the zipped sprint submissions and demo videos[cite: 1280, 1284].
* [cite_start]`/docs`: Contains all project planning [cite: 2][cite_start], requirements [cite: 225][cite_start], and design documents[cite: 240].
* `/src`: Contains all Java/C# source code.
    * `/data`: Database connection and query logic.
    * `/model`: Data classes (Customer, Order, Pizza, etc.).
    * `/ui`: All GUI screens (Login, Menu, Cart, etc.).
